* HOW TO RUN
This is a Python 3 project, and should be explicitly run with =python3=. Simply run the following and the GUI will open:
#+BEGIN_EXAMPLE
python3 main.py
#+END_EXAMPLE
If you only have Python 3 installed, then you should just use:
#+BEGIN_EXAMPLE
python main.py
#+END_EXAMPLE
